🎯 Propósito
Digitalizar y centralizar los procesos clínicos y administrativos, optimizando la gestión de citas, historiales médicos y comunicación entre pacientes y personal de salud.

🏗️ Arquitectura
text

├── 🖥️ Backend (API REST)
│   ├── Python + FastAPI (async)
│   ├── MySQL 8.0 (base de datos)
│   └── Modelos de IA (clasificación de texto)
│
└── 📱 App Móvil (Android)
    ├── Kotlin (nativo)
    └── Android SDK 34
