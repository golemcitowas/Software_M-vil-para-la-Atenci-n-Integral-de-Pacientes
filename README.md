El sistema está compuesto por un backend desarrollado en Python con FastAPI que proporciona una API REST asíncrona para la gestión de citas médicas, disponibilidad de especialistas, autenticación de usuarios con JWT, historial clínico, y clasificación de urgencias mediante modelos de inteligencia artificial con Transformers y PyTorch; y una aplicación móvil nativa para Android desarrollada en Kotlin que permite a los pacientes solicitar citas, consultar especialidades disponibles, visualizar su historial médico y recibir atención personalizada. La base de datos principal es MySQL 8.0 y se integra con Supabase para la autenticación y almacenamiento en la nube, brindando escalabilidad y sincronización en tiempo real. El servidor Uvicorn ejecuta la API en el puerto 8000 con documentación automática disponible en /docs. El proyecto se encuentra actualmente en fase de desarrollo activo.
